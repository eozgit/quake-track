Issue tracking system, built with:

  • .Net Core 3.1 Web API
  • C#
  • Angular 8
  • TypeScript
  • NgRx
  • Bootstrap
  • Swagger
  • OpenAPI

Code is available on Github

API specs are available on Swaggerhub

Features:

  • Local and Google authentication
  • CRUD Projects
  • CRUD Issues
  • Kanban board
